Install the Edge101 Onboarding Image

Connect your Edge101 device by USB and flash the published PVAutonomy Onboarding Image directly from this page. The same image is used for PVAutonomy Managed and the Open-Source path — no account, no payment, no repository access, and no keys are required.

This is a technical page, not a finished product. It has not been approved as Customer-Ready or Stable and has not been validated on real hardware. It makes no claim about a successful flash, full erase, Wi-Fi setup, captive portal, or Home Assistant adoption.

1 · Flash over USB

This browser did not provide the Web Serial API on this page, so in-browser installation is unavailable here. You can download the image and flash it yourself using the links in section 3 below. In-browser installation requires a secure context (HTTPS or localhost). You can download the image manually in section 3 below.

The Web Serial API is not available in this environment. Use the manual download in section 3.

In-browser installation uses the Web Serial API. If your computer does not detect a serial device, you may need a USB-to-UART driver: CP210x (Silicon Labs) · CH340 (WCH).
